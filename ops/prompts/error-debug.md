# Prompt para diagnóstico de erro

Analise este erro do OPEN-NOTEBOOK.

Contexto:
- macOS
- Docker Compose
- Repositório: https://github.com/lfnovo/open-notebook.git

Antes de sugerir correção:
1. Identifique a causa raiz provável.
2. Liste evidências do log.
3. Diga quais arquivos precisam ser lidos.
4. Separe hipótese de fato confirmado.
5. Não sugira comando destrutivo.
6. Não apague volumes.
7. Não altere docker-compose.yml sem explicar impacto e rollback.

Erro/log:

```text
COLE_AQUI_O_LOG_COMPLETO
```

Entregue:
- Diagnóstico
- Causa provável
- Comandos seguros de inspeção
- Correção recomendada
- Riscos
- Como validar
- Como evitar no futuro
